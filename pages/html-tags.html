Intro

  • HTML determines the structure of web pages:
    • Hypertext - text (often with embeds such as images, too) that is organized in order to connect related items
    • Markup - a style guide for typesetting anything to be printed in hardcopy or soft copy format
    • Language - a language that a computer system understands and uses to interpret commands
  • Generally, contained in a file with .html extensions, or when returned by a server as a web page has response header content-type: text/html; charset=utf-8
  • Used with assisted technologies such as CSS and JavaScript to make HTML beautiful and add interactivity, respectively
  • HTML tag - string of a specific format that defines a block or piece of HTML file content, most HTML tags have an opening tag containing the tag name, tag attributes, a closing tag containing a forward slash, and the tag name being closed, or tags that do not have a closing tag like <img>, it is best practice to end the tag with a forward slash

    <!doctype html>
    <html>
    <head>
      <meta charset="utf-8">
      <title>Example page</title>
    </head>
    <body>
      <h1>This is a heading</h1>
      <p>This is an example of a basic HTML page.</p>
    </body>
    </html>

Links

  • target attribute specifies where to open the linked page/document:
    • _blank - Opens the linked document in a new window or tab
    • _self - Opens the linked document in the same window/tab as it was clicked (this is default)
    • _parent - Opens the linked document in the parent frame
    • _top - Opens the linked document in the full body of the window
    • framename - Opens the linked document in a named frame
  • use schema for specific action (<a href="schema:path">txt</a>):
    • http://example.com
    • mailto:<email>[,<email>]*[?<arguments>]
    • tel:+3112345678
    • sms:<phone_number>[,<phone_number>]*[?body=<message_body>]
    • geopoint:latitude,longitude
  • Scroll-to-text fragments (also known as text fragments) - append a text query to a URL, find the text in the webpage, scroll it into view, and highlights the matched text: #:~:text=\[prefix-,]textStart[,textEnd\][,-suffix], example:
    • https://en.wikipedia.org/wiki/History_of_computing#:~:text=By%20the%20late%201960s

filepath

  • absolute path (full URL to an internet file, recomended way !): src="http://www.example.com/images/picture.jpg"
  • relative path (relative to the current page):
    • src="picture.jpg" - picture.jpg is located in the same folder as the current page
    • src="../images/picture.jpg" - picture.jpg is located in the images folder in the current folder
    • src="/images/picture.jpg" - picture.jpg is located in the images folder at the root of the current web
    • src="../picture.jpg" - picture.jpg is located in the folder one level up from the current folder
